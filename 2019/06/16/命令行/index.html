命令行

发表于 2019-06-16 |

于命令行运行包下java程序的学习记录的同时，记录些关于命令行与键盘的快捷键。持续更。

快速打开命令行

win+r+cmd+回车

全屏

Alt+回车

改变cmd前景色和后景色

输入：color 0x 0是背景色，x是字体颜色

参考：0 黑色，1 蓝色，2 绿色，3 浅绿色，4 红色，5 紫色，6 黄色，7白色，8 灰色，9 浅蓝，A 浅红，B 浅蓝色，C 浅红色，D 浅紫色，E 浅黄色，F 亮白色。（十六进制嘛~ 字母大小写不影响 附加代码滚动：dir/w/s,Ctrl+c停止）

包下java程序的编译与运行

转到目录
编译：javac -d . xxx.java (-d表示在当前目录)
运行: java 包名.xxx
